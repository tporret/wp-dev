== Description ==

Quickly add scripts, styles or fonts to your site with ease. 
Once added it's available on both the admin side and front facing pages.

Always looking for more ideas but I made this plugin because I use it on almost every project!

Combine it with Elementor Extension - Three Panels to get HTML, CSS & JS sections on any elementor page.

== Frequently Asked Questions ==

= How do I use it =

Just name and add your asset url, and select if it should be in the footer or not.
That's It! You can now use that asset anywhere on the site.
